# SAFE-MFI Mapping to the NIST AI Risk Management Framework (AI RMF 1.0)

## Overview

The NIST AI Risk Management Framework (AI RMF 1.0) provides guidance for designing, developing, deploying, and managing trustworthy AI systems. SAFE-MFI operationalizes these principles through lifecycle engineering activities integrated into the machine learning workflow.

## Mapping

| NIST AI RMF Function | SAFE-MFI Implementation                                                           |
| -------------------- | --------------------------------------------------------------------------------- |
| Govern               | AI governance documentation, Model Cards, Audit Logs, AI Impact Assessment        |
| Map                  | Data profiling, stakeholder analysis, risk identification                         |
| Measure              | Accuracy, Precision, Recall, F1-score, ROC-AUC, SHAP, Fairness Metrics            |
| Manage               | Drift monitoring, retraining workflow, deployment checklist, governance dashboard |

## Trustworthiness Characteristics

SAFE-MFI supports:

* Validity and reliability
* Safety
* Security and resilience
* Explainability
* Privacy-aware design
* Fairness
* Accountability
* Human oversight

## Summary

SAFE-MFI embeds NIST AI RMF activities directly into the machine learning lifecycle, enabling continuous AI risk management rather than one-time model validation.
